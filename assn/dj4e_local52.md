Installing Django 5.2 Locally
=============================

**Note:** The `dj4e_local52` material (this page and the per-platform guides it links to) is **experimental**. It is under active **testing and development**; steps or tooling may change. If something does not match your environment, use the main course install paths or ask on the course forum.

Use the guide for **your** operating system. Each walks through running Django 5.2 on your own computer and exposing it with <a href="https://localhost.run" target="_blank">localhost.run</a> so DJ4E autograders can reach your site.

| Platform | Guide |
| -------- | ----- |
| **Windows** | [dj4e_windows52.md](dj4e_windows52.md) |
| **WSL** (Linux on Windows) | [dj4e_wsl52.md](dj4e_wsl52.md) |
| **Mac** | [dj4e_mac52.md](dj4e_mac52.md) |
| **Linux** | [dj4e_linux52.md](dj4e_linux52.md) |

When you finish, you will have a localhost.run URL (for example `https://xxxxx.lhr.life`) to submit to the Install autograder.

**Troubleshooting:** [Fixing Common Django Errors](dj4e_errors52.md)

**Django 4.2:** As of January 2026 this course uses Django 5.2. For 4.2, see [dj4e_install42.md](dj4e_install42.md) and adapt the steps from your platform guide above.

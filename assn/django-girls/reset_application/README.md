# Reset Your Application

If your Django Girls project has gotten into a bad state and you want to start over from scratch, you can reset by deleting the entire `djangogirls` folder and beginning again.

## Commands to reset

In the PythonAnywhere **Bash** console, run:

```bash
cd ~
rm -rf ~/djangogirls
```

This removes the entire `djangogirls` folder and everything inside it.

## Start fresh

After deleting the folder, go back to the beginning.

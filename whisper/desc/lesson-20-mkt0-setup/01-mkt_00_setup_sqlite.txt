Django Marketplace Initial Project Setup with Virtual Environment

Django, virtual environment, WSGI, marketplace, migrations, superuser

This lecture details the initial configuration steps for the Django Marketplace project. It covers creating a new Django project named Market with a configuration directory structure, setting up a Python 3.13 virtual environment, and installing Django 5.2. Key steps include generating and applying migrations, creating a superuser with the username admin, and configuring WSGI to switch from the MySite project to Market.

After completing these steps, the Marketplace application runs locally and is accessible via the web server. The setup includes an admin interface for user management and provides the foundation for developing the Marketplace features as specified in the assignment.

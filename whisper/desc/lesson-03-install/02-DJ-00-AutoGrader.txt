Django Autograder: How It Checks Student Assignments

autograder, django, php, lti, html, web crawling, grading

The Django autograder is a PHP program hosted at www.dj4e.com that checks student assignments by sending HTTP requests to the student's website and verifying the HTML response contains specific strings as defined by the assignment instructions.

Students must manually test their application before submitting it to the autograder, which uses Learning Tools Interoperability to report grades. The autograder runs up to 50 request-response cycles to validate the application and provides detailed feedback on mismatches if the assignment requirements are not met.

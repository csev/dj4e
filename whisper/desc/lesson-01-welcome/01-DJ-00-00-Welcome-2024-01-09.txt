Django for Everybody Introduction

python, django, object-oriented programming, html, css, javascript, json, oauth, bootstrap, pythonanywhere

Charles Severance has taught web development courses at the University of Michigan since 2007, using frameworks including PHP, Ruby on Rails, Google App Engine, and Flask. He developed the popular Python for Everybody course for beginners.

The Django for Everybody course teaches advanced Python and object-oriented programming by building a complete website with features such as Ajax, image handling, OAuth authentication, and Bootstrap, hosted on Python Anywhere.

Basic HTML Document Structure and Syntax

html, head, body, relative links, absolute links, anchor tag, image tag, special characters, comments, unordered list, table

The lecture covers the basic structure of an HTML document, starting with the `html` root element that contains the `head` and `body` sections. The `head` holds non-printing information like the page title, styles, and scripts, while the `body` contains the visible content. It introduces common tags such as `p` for paragraphs, `strong` for bold text, and `a` for hyperlinks.

The lecture explains entity references for special characters (e.g., `&lt;` for `‹`), HTML comments (`‹!--`), and the distinction between relative links (e.g., `href` within the same directory) and absolute links (full URLs). It also describes unordered lists using `ul` and `li` tags, tables with `table`, `tr`, `th`, and `td` elements, and the `pre` tag for preserving whitespace and line breaks.

Basic HTML Document Structure and Tags

html, head, body, tags, relative links, absolute links, img, a, ul, li, table, tr, th, td

The lecture explains the fundamental structure of an HTML document, starting with the doctype (optional), followed by the `html` element that contains the `head` (for non-printing metadata) and `body` (for printable content). The `head` section includes the page title, styles, and scripts.

Key concepts covered include the use of `img` for images, `a` for hyperlinks with `href` attributes (supporting relative and absolute URLs), `ul` and `li` for unordered lists, and `table`, `tr`, `th`, `td` for tabular data. The lecture emphasizes that tables should be used for tabular data and not for layout, and that HTML focuses on semantic meaning rather than visual presentation.

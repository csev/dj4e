HTML Basic Syntax and Document Structure

html, document structure, head, body, hyperlinks, relative links, absolute links, image tags, list tags

The lecture covers the foundational structure of an HTML document, including the doctype declaration, the <html> root element, the <head> section for metadata and non-printing content, and the <body> section for visible page content. It emphasizes that the <head> contains elements such as the page title, styling information, and scripts, while the <body> holds the actual content that users see.

The lecture details basic HTML tags and their attributes, including hyperlinks (using the <a> tag with href attributes), image tags, and unordered lists (with <ul> and <li> tags). It also explains HTML special characters and their entity representations (e.g., &lt; for less than), the use of HTML comments, and the distinction between absolute and relative links for resource references, as well as the web server's default file naming conventions (e.g., index.html).

Three Methods for Applying CSS in HTML

CSS, HTML, style attribute, style tag, external style sheet, cascading, selector, id, class, span, div

This lecture explains the three primary methods for applying CSS rules in HTML: using the style attribute directly on elements, placing CSS in a style tag within the head section, and linking to an external CSS file. It also covers cascading behavior where closer rules have higher priority, and introduces `span` and `div` as elements with no default styling for precise visual control.

The lecture details CSS selectors including `id` (unique per document) and `class` (reusable across elements), and how hierarchical selectors allow targeted styling without excessive classes. These techniques enable fine-grained control over specific elements within HTML structures while maintaining clean, maintainable code.

Django for Everybody: Linux Command Line Basics

Bash, Command line, Django shell, Linux, SQLite, Python shell

This lecture introduces the Linux command line interface as the primary environment for server operations, focusing on Bash shell commands essential for Django development. It covers basic navigation with `ls`, `pwd`, and `cd`, file manipulation using `rm`, text search via `grep`, and documentation through `man`. The session also explains command line completion using the tab key and the history feature for re-executing previous commands.

The lecture distinguishes between the Linux shell, Python interactive shell (REPL), and Django shell, highlighting how to recognize and exit each environment. It demonstrates SQLite database interaction and practical commands for Django development, such as running the Django shell with `python manage.py shell` and using the `quit` command to exit.

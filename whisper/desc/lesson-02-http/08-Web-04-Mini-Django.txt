Mini-jango: Under-200-line HTTP server

mini-django, http server, url routing, views, django

Mini-jango is a minimal implementation of Django under 200 lines of Python. It provides a simplified HTTP server for understanding the internal structure of Django.

The framework processes HTTP requests through a URL router that directs them to specific view functions. Each view function receives an HTTP request and returns an HTTP response, with the system handling common errors like 404 not found and 500 internal server errors.

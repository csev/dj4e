Inheritance in Object-Oriented Programming

inheritance, parent class, child class, subclass, code reuse

Inheritance is a mechanism in object-oriented programming where a new class (child) inherits attributes and methods from an existing class (parent). This allows developers to reuse existing code without duplicating functionality.

The lecture demonstrates inheritance with a football fan class that extends the party animal class. The football fan class inherits all attributes and methods of the party animal class and adds a points attribute and a touchdown method that increments the points by seven.
